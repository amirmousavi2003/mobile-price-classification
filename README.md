
پیاده‌سازی و تنظیم پارامتر دو الگوریتم Decision Tree و KNN روی دیتاست Mobile Price Classification، با استفاده از پایتون (scikit-learn) و نرم‌افزار RapidMiner.

این پروژه برای آشنایی عملی با دو الگوریتم پایه‌ی طبقه‌بندی ساخته شده:

Decision Tree — مدلی مبتنی بر قانون که داده‌ها را بر اساس آستانه‌ی هر ویژگی تقسیم می‌کند
KNN — مدلی مبتنی بر فاصله که هر نمونه را بر اساس نزدیک‌ترین همسایه‌هایش طبقه‌بندی می‌کند

اهداف اصلی پروژه:

ارزیابی مدل‌ها با معیارهای استاندارد (Accuracy، Precision، Recall، F1-score، ماتریس درهم‌ریختگی)
مقایسه‌ی همان آزمایش‌ها در دو ابزار متفاوت: پایتون و RapidMiner

📊 دیتاست
Mobile Price Classification — شامل ۲۰۰۰ نمونه گوشی موبایل با ۲۰ ویژگی (باتری، رم، اندازه صفحه، مشخصات دوربین و...) و یک ستون هدف به نام price_range با ۴ کلاس (۰ = ارزان، ۳ = خیلی گران).
⚙️ نحوه‌ی کارکرد
پایپ‌لاین پایتون (mobile_price.py)
۱. دیتاست را بارگذاری و به نسبت ۷۰ درصد آموزش و ۳۰ درصد آزمون تقسیم می‌کند
۲. ویژگی‌ها را با StandardScaler نرمال‌سازی می‌کند
۳. مدل Decision Tree را با چند max_depth و دو معیار gini و entropy آموزش می‌دهد
۴. مدل KNN را با چند مقدار K و دو متریک euclidean و manhattan آموزش می‌دهد
۵. برای هر ترکیب، Accuracy، Precision، Recall، F1-score و فاصله‌ی دقت Train/Test را محاسبه می‌کند
۶. جدول‌ها و نمودارها را در پوشه‌ی outputs ذخیره می‌کند
پروسه‌ی RapidMiner (فایل .rmp)
همان پایپ‌لاین را به‌صورت بصری پیاده می‌کند:
Retrieve → Numerical to Polynominal → Split Data (۷۰/۳۰) → Decision Tree / k-NN → Apply Model → Performance
